    What is OData?

    • OData is a standardized protocol for creating and consuming data-centric apis, building on HTTP and REST. See http://www.odata.org/

    What is odata4j?

    • odata4j is a Java framework that implements the OData protocol for both consumers (client-side) and producers (server-side)

    Getting started: Consuming OData

    • Download the latest stable distribution archive
    • Add odata4j-clientbundle-x.x.jar from distribution archive to your build path
    • Use the odata4j consumer api to talk to an existing OData service
    • For example...
    // create consumer instance
String serviceUrl = "http://services.odata.org/OData/OData.svc/";
ODataConsumer consumer = ODataConsumers.create(serviceUrl);

// list category names
for (OEntity category : consumer.getEntities("Categories").execute()) {
  String categoryName = category.getProperty("Name", String.class).getValue();
  System.out.println("Category name: " + categoryName);
}

    Getting started: Producing OData

    • Download the latest stable distribution archive
    • Add odata4j-bundle-x.x.jar from distribution archive to your build path (or odata4j-nojpabundle-x.x.jar for a smaller bundle with no JPA support)
    • Choose or implement an ODataProducer using the odata4j producer api
    • For example...
      • Use InMemoryProducer to expose POJOs as OData entities using bean properties: InMemoryProducerExample.java
      • Use JPAProducer to expose an existing JPA 2.0 entity model: NorthwindJPAProducerExample.java
      • Implement the ODataProducer interface directly
      • Expose AppEngine's Datastore as an OData endpoint: odata4j-appengine
      • Host your producer in Tomcat: http://code.google.com/p/odata4j/wiki/Tomcat

    Framework design principles

    • Leverage existing Java standards (e.g. JAX-RS, JPA, StAX)
    • Utilize a small number of carefully-chosen Java open-source components (e.g. jersey, Joda Time)
    • Support OData consumers in constrained Java client environments, specifically Android
    • Support OData producers in constrained Java server environments, specifically Google AppEngine
